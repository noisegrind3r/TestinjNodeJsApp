# TestNodeJsApp

Программа app.js
Тесты jest app.test.js

Есть хомяки (-hamster) следующих видов
	- Хомяк обычный (common)
	- Хомяк ангорский (angora)
	- Хомяк джунгарский (djungarian)
	- Дикий лесной хомяк (wild)
	- Резиновый хомяк (rubber)

У них есть свойства (-property)
	- Имя (name)
	- Еда, которую он ест (food)
	- Скорость вращения колеса (speed)
	- Громкость верещания (volume)

Они умеют (-action)
	- Есть (eat)
	- Крутить колесо (roll)
	- Пищать (vocalize)
